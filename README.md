# Random Quote Generator

This is a simple web application built using HTML, CSS, and JavaScript.

## Features
- Displays a random quote when the app loads.
- "New Quote" button shows a different quote each time.
- Shows both quote text and author name clearly.
- Clean and minimal lavender-themed user interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the repository.
2. Open index.html in any web browser.
